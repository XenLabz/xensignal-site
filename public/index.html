What XenSignal does

XenSignal evaluates candidate entries using model fair value, market pricing, spread, timing, position state, and optimizer guidance.

Why shadow mode matters

Shadow mode lets the system observe, score, and log candidate trades without blocking or placing real orders. The machine gets to prove itself before touching money. Radical concept, somehow.

Current operating posture

Paper-watch only. Engine consumption disabled.

The optimizer can recommend profiles and filters, but live execution blocking is intentionally not enabled.
